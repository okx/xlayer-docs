# Ankr

[Ankr](https://www.ankr.com/rpc/xlayer/ "Ankr") simplifies the process of setting up and participating in Proof of Stake (PoS) blockchains. It provides various tools and services to help users deploy nodes, stake their PoS tokens, and access decentralized finance (DeFi) applications.

**Supported network**: 
- X Layer mainnet
